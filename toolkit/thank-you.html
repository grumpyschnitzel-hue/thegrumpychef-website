Getting Started

Recommended Implementation Order

Don't try to use all 8 at once. Start with the foundation, then layer on.

Recipe Costing First
Open Template 01. Cost your top 10 dishes. This is the foundation — every other template builds on knowing your plate costs.
Menu Engineering Matrix Second
Once you have plate costs, open Template 03. Enter your item sales data. The matrix shows you exactly which dishes to keep, cut, reprice, or promote.
Weekly Prime Cost Tracker Third
Open Template 02. Start tracking weekly. This is the number that tells you if you're making money — before the bank statement does.
Add the Rest As Needed
Par levels, waste tracking, delivery margins, menu psychology, and vendor comparison — layer these in as you stabilize your core numbers.

How the Templates Work

Yellow cells — your inputs (type here)
Gray cells — auto-calculated (don't touch)
Green — on target
Red — needs attention

Using Google Sheets? Upload the .xlsx file to Google Drive, then right-click → Open With → Google Sheets. All formulas and formatting carry over.

## LinkedIn Carousel Creation Tools

### Recommended Stack (Solo Creator, High Volume)

**Primary design tool:** Canva (with Carousel Studio + brand kit templates)
- Drag-and-drop, brand kit integration, AI-powered "Carousel Studio" generates multi-slide designs from a topic
- Best for: consistent branding, speed, minimal learning curve

**Quick-draft AI tools (optional boosters):**
- **aiCarousels.com** — Free, no signup. Enter topic or paste content → AI drafts slide copy + layouts. Built-in editor with auto-resizing.
- **Reepl** — Free, no signup or credit card. Drop a topic, URL, YouTube link, or PDF → auto-builds carousel slides.
- **CarouselMaker (carouselmaker.co)** — Free AI carousel maker. Drag-and-drop reorder, randomize fonts/colors.
- **Taplio** — Free generator. Turns tweets, threads, Reddit posts, or scratch topics into carousel drafts.
- **Contentdrips** — Always-free basic generator. Pro adds AI content and more templates.

**Agency-level design:** DocHipo — polished pre-built LinkedIn carousel templates, professional output without a designer.

**Advanced/custom:** Figma — more control, higher learning curve, better for custom design systems.

### Production Workflow (Recommended)
1. **Draft:** Use Reepl or aiCarousels to generate first draft from topic or existing text
2. **Polish:** Import slides into Canva, overlay brand kit (colors, fonts, logo)
3. **Export:** Save as multi-page PDF (1080×1080 or 1080×1350 per page)
4. **Schedule:** Upload to LinkedIn scheduler for optimal time slots (Tue–Thu mornings)

## How to Upload a PDF Carousel on LinkedIn

### Desktop
1. Go to LinkedIn home feed → click "Start a post"
2. Click "More" (three dots) → select "Add a document"
3. Click "Choose file" → select your multi-page PDF
4. Wait for upload — LinkedIn shows the page count
5. Add a **document title** — this appears above the carousel in the feed (treat it like a hook)
6. Click "Done"
7. Write your post caption above the document: hook in first 1–2 lines, 2–4 lines context, CTA
8. Add 2–5 hashtags
9. Hit "Post"

### Mobile
1. Open LinkedIn app → tap "Post" / "Create post"
2. Tap "Add a document" icon (under "More")
3. Choose PDF from phone or cloud service
4. Add document title
5. Write caption (hook, context, CTA) + hashtags
6. Tap "Post"

### Common Mistakes
- Uploading individual images instead of one PDF → creates an image grid, not a carousel
- Tiny or excessive text per slide → unreadable on mobile
- Weak document title → people see this before they swipe

## Taplio Carousel Generator Tutorial

### Method A: Repurpose Existing Content
1. Go to taplio.com/carousel
2. Copy URL of a Twitter/X thread or Reddit post
3. Paste URL into the generator field
4. (Optional) Click "Advanced options" → pick theme, toggle engagement display
5. Click "Generate Carousel" → wait a few seconds
6. Review slides, adjust theme if needed
7. Click "Download Carousel"

### Method B: Create from Scratch with AI
1. Go to Taplio's AI carousel editor
2. Pick a template from the gallery
3. Enter topic (e.g., "7 systems to fix a small restaurant kitchen")
4. Let AI generate first draft
5. Edit slide text to match your voice
6. Add profile picture, name, handle for branding
7. Adjust theme, colors, fonts
8. Preview → download as PDF or images

### Posting via Taplio Scheduler
1. Log into Taplio → "Create Post"
2. Choose "Carousel" → upload downloaded images/PDF
3. Write caption (hook + context + CTA) + hashtags
4. Preview → Publish or Schedule

## Instagram Reels Production

### Recording Setup
- **Orientation:** Always vertical 9:16
- **Resolution:** 1080p minimum, 4K preferred
- **Audio:** External mic if possible — clean audio is non-negotiable
- **Lighting:** Natural or soft diffused light, stable framing
- **Editing pace:** Cut every 1.5–2 seconds to avoid dead air

### Tools for Reels
- **Editing:** CapCut (free, mobile-first), InShot, or native Instagram editor for simple cuts
- **Captions/Subtitles:** CapCut auto-captions or Submagic for burned-in subtitles
- **Scheduling:** Later, Buffer, or Meta Business Suite for scheduling Reels in advance

### Common Mistakes
- Re-uploading TikToks with visible watermarks (algorithm deprioritizes)
- Text outside the safe zone (covered by UI elements)
- No subtitles (many users watch on mute)
- Inconsistent posting cadence

## Instagram Carousel Creation

### Tools
- Same tools as LinkedIn carousels (Canva, aiCarousels, etc.)
- **Dimensions:** 1080×1080 (square) or 1080×1350 (portrait, more screen real estate)
- Export as individual images (not PDF — Instagram uses image uploads, not documents)

### Upload Process
1. Open Instagram → tap "+" → select "Post"
2. Tap the "multiple images" icon (overlapping squares)
3. Select slides in order (up to 20 images)
4. Apply consistent filter if desired (same filter across all slides)
5. Write caption with hook, context, and CTA
6. Add relevant hashtags (5–15 for Instagram)
7. Post or schedule via Meta Business Suite

## Cross-Platform Repurposing Workflow

### The Pillar Content System
1. **Create 1–2 pillar pieces per month** (deep framework, case study, complete guide)
2. **Extract 10–15 pieces per pillar** by pulling out: lessons, stories, data points, quotes, questions
3. **Adapt per platform** (not copy-paste — adapt format and tone)

### One Idea → Multiple Platforms
From a single core idea, produce:
- **Instagram Reel:** Story/demo or before-after using the idea (20–30 sec)
- **IG Carousel:** Step-by-step or "X mistakes / X fixes" (5–7 slides)
- **Twitter/X single tweet:** Sharp take or stat (280 chars)
- **Twitter/X thread:** Story + lesson + action steps (5–7 tweets)
- **LinkedIn text:** Full framework post (800–1,600 chars)
- **LinkedIn carousel:** Expanded breakdown (5–10 slides)
- **Newsletter section:** Deeper story and playbook with CTA
- **Substack Note:** Quick insight or question from the idea
- **Facebook group post:** Peer story + question

### Weekly Pillar Stretch
- **Week 1:** Overview post (pillar intro), big story Reel, LinkedIn text
- **Weeks 2–3:** Deep dives on sub-topics, each with IG + X + LinkedIn content
- **Week 4:** Case study Reel + newsletter tying everything together

### 90/10 Content Mix Rule
Apply across all platforms:
- **90% value:** Frameworks, stories, transformation proof, answering questions, engagement — no strings attached
- **10% pitch:** Scorecard download, tool CTAs, discovery call offers, system-focused promotions

For 30 posts/month: 27 value, 3 pitch.
For 5 posts/week: ~4-5 value, pitch woven into 1 post max.

## Canva Export Settings for LinkedIn Carousels

### Recommended Settings
- **Dimensions:** 1080×1080 (square) or 1080×1350 (portrait, more screen real estate)
- **Format:** Export as PDF (Standard quality is fine)
- **File naming:** Use the carousel topic for easy retrieval (e.g., "7-kitchen-systems-carousel.pdf")
- **Font size minimum:** 24pt for body text, 36pt+ for headlines (mobile readability test)
- **Max 3 colors** from your brand kit
